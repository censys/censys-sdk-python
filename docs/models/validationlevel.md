# ValidationLevel

The extent to which the certificate's issuer validated the identity of the entity requesting the certificate. Options include Domain validated (DV), Organization Validated (OV), or Extended Validation (EV).


## Values

| Name      | Value     |
| --------- | --------- |
| `UNKNOWN` | unknown   |
| `DV`      | dv        |
| `OV`      | ov        |
| `EV`      | ev        |
